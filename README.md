# Emergency Info BC Plugin

## Description
Provides custom functionality for the Emergency Info BC site.

## Setup
1. Activate Custom Post Type UI and Advanced Custom Fields plugins.
1. Import the json file(s) in imports/custom-post-type-ui/ through CPT UI's import tool.
1. Import the json file(s) in imports/advanced-custom-fields/ through ACF's import tool.
